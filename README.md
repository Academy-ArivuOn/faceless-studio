# 🎬 Faceless YouTube Operating System (AI SaaS)

An AI-powered multi-agent system that transforms a simple niche input into a complete, **upload-ready content package** for faceless YouTube creators.

> From idea → script → scenes → SEO → publishing plan — all in one flow.

---

## 🚀 Live Concept

Input:

```
Niche: Personal Finance for Students  
Platform: YouTube  
Tone: Educational  
```

Output:

* 5 Viral Video Ideas
* Full Script (Hook + Body + CTA)
* Scene-by-Scene Breakdown
* Hook Psychology Explanation
* YouTube Title + Description + Tags
* Shorts Script (60s)
* Instagram & TikTok Captions
* 7-Day Content Posting Plan

---

## 🧠 Core Idea

Most AI tools give **text output**.

This system delivers a **complete execution workflow**.

> Not a prompt tool. Not a caption generator.
> A full **Content Operating System** for creators.

---

## ⚙️ Architecture

### 🔁 Multi-Agent Pipeline

```
User Input
   ↓
[Research Agent]
   ↓
[Creator Agent]
   ↓
[Publisher Agent]
   ↓
Final Output (Upload-ready content)
```

---

### 🧩 Agents Overview

#### 🔍 Research Agent

* Finds trending topics
* Identifies competitor gaps
* Generates high-converting hooks

#### ✍️ Creator Agent

* Writes full script
* Generates scene breakdown
* Adds voiceover instructions
* Explains hook psychology

#### 📢 Publisher Agent

* Creates SEO titles & descriptions
* Generates tags & hashtags
* Produces cross-platform content
* Suggests best posting time

## ⚡ Getting Started

### 1. Clone the Repository

```
git clone https://github.com/your-username/faceless-youtube-os.git
cd faceless-youtube-os
```

---

### 2. Install Dependencies

```
npm install
```

---

### 3. Setup Environment Variables

Create `.env.local`:

```
GEMINI_API_KEY=your_gemini_api_key
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
```

---

### 4. Run the App

```
npm run dev
```

App runs at:

```
http://localhost:3000
```
